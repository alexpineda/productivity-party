## Getting Started

1. Install dependencies: `npm install`
2. Start development servers:
   - Next.js: `npm run dev`
   - PartyKit: `npm run partykit:dev`
   - Both: `npm run dev:all`
3. Access the debug tools at `/debug` to monitor server health and status
